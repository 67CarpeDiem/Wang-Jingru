IntelliJ IDEA 2022.3 最新变化

在 IntelliJ IDEA 2022.3 中，可以通过设置切换到新 UI，即可预览焕然一新的 IDE 外观。 此版本引入了一个新的 Settings Sync（设置同步）解决方案，用于同步和备份自定义用户设置。 此外，IntelliJ IDEA Ultimate 还提供了一种处理 WSL 2 中的项目的新方式、适用于 Spring Bean 自动装配和 OpenAPI 架构生成的新操作，以及备受期待的 Redis 支持。 新版本的 IDE 具有多项其他改进和升级，下面将详细介绍。 下载

我们实现了一系列新的 Java 检查和快速修复，保护您的代码免受潜在危害和错误。 新增了一项检查来帮助检测在每个分支中都有一条公共语句的 switch 表达式，并提供了一个快速修复来将语句向上移动到 switch 表达式中，从而缩短代码。 IDE 将报告冗余数组长度检查，以及 Collection.parallelStream() 之后的冗余 parallel() 调用。 另一项新检查可以报告仅使用一个元素或字符时数组、列表或字符串的多余创建。 您可以在这篇博文中详细了解 IntelliJ IDEA 2022.3 中的新检查和快速修复。

IntelliJ IDEA 2022.3 支持 2022 年 9 月发布的 Java 19 的新功能。 IDE 现在支持记录模式以及对 switch 表达式模式匹配的更改，提供了代码高亮显示、补全和导航。 现有检查和快速修复已相应更新以支持这些更改。 有关详情，请参阅此博文
